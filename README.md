# Nazirite Beard Oil Co. – MERN E-Commerce Store

A full-stack e-commerce platform built with the **MERN stack** for **Nazirite Beard Oil Co.** This project is being developed as a real online storefront for a beard oil brand while also serving as a portfolio-quality full-stack application.

Nazirite Beard Oil Co. is focused on premium beard care products with clean branding, quality ingredients, and a simple shopping experience.

## About the Brand

Nazirite Beard Oil Co. is a beard care brand built around premium beard oils designed to help keep beards healthy, soft, and well-groomed.

**Clean ingredients. Bold scents. Healthy beards.**

## Features

- Product catalog
- Product detail pages
- Shopping cart functionality
- Ratings and reviews
- Inventory tracking
- Responsive design
- Admin management features
- Full-stack MERN architecture
